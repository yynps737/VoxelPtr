# Changelog

All notable changes to VoxelPtr will be documented in this file.

## [1.0.0+mc1.20.4] - 2025-11-21

### ✨ Added
- **实时矿石扫描系统**
  - 自动扫描周围区块的矿物
  - 智能缓存机制，每个区块只扫描一次
  - 异步执行，不阻塞游戏主线程

- **10 种矿物预设快速切换**
  - 支持钻石、铁、金、绿宝石、远古残骸、煤、红石、青金石、铜、石英

- **HUD 实时显示**
  - 显示最近的矿石位置和距离
  - 三维方向指示（上下、左右、前后）
  - 支持 4 个 HUD 位置，彩色文字显示

- **高度可配置**
  - 扫描半径：1-10 区块
  - 显示数量：1-50 个目标
  - 实时配置更新，无需重启游戏

- **完整国际化支持**
  - 中文 (zh_cn) / 英文 (en_us)

### ⚡ Performance
- 距离计算优化（提升 40-50%）
- 颜色查询优化（提升 70-80%）
- 方向计算缓存（提升 60%）
- **整体性能提升 50-70%**

### 🎮 Controls
- `V` 键 - 启用/禁用矿石追踪
- `N` 键 - 切换矿物预设
